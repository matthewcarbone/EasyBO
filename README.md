# EasyBayes

⚠️ **This is a work in progress and breaking changes will be common**

EasyBayes is a collection of tutorials, resources and code that hopefully will make my life easier when doing probabilistic modeling, and running Bayesian optimization experiments. Maybe it'll make your life easier too!
